Git-Kommandos

Command Beschreibung Info
git add * Änderungen werden vorgeschlagen. Durch den Stern werden alle Dateien im Verzeichnis hochgeladen. Statt des Sternchens können auch gezielt Dateien genannt werden, die hochgeladen werden sollen.
git commit -m "Nachricht" Änderungen werden bestätigt. In der Nachricht dann eine verständliche Erklärungen für die anderen Entwickler*innen hinterlassen, welche Änderungen in dieser Version durchgeführt wurden.
git push Änderungen werden zu GitHub hochgeladen.
git branch Übersicht: Welche Branches haben wir und in welchem Befinde ich mich?
git branch branchname Erstellt einen neuen Branch. Bei branchname immer den gewünschten (neuen) Branch einfügen. Der neue Branch muss dann noch gepushed werden (siehe unten).
git checkout -b branchname Erstellt einen neuen Branch und wechselt sofort in diesen Branch.
git push -u origin branchname Lädt den neuen Branch hoch.
git checkout branchname Wechselt zu dem gewünschten Branch.
git branch -d branchname Löscht den Branch im lokalen Repository
git checkout main In der Hauptastamm wechseln
git pull Um den aktuellen Stand des des Masters/der Branch in Visual Studio zu laden. Darauf achten, in welcher Branch man sich befindet.
git merge branchname Um den Arbeitsstand eines Branches in den Master zu implementieren. Um den Hauptstamm zu ändern, muss man sich im Master befinden.
git merge origin/master Um in die eigene Branch die aktuelle Version aus dem Master zu laden. Dafür muss man sich dann auch in der eigenen Branch befinden.

Terminal-Kommandos

Command Beschreibung Info
cd "Change Directory" - wechselt in einen Unterordner
pwd "Print Working Directory" - zeigt das aktuelle Verzeichnis an
ls "List" - zeigt den Inhalt eines Ordners an ls -a zeigt auch unsichtbare Dateien an
cp "Copy" - kopiert Datei Beispiel: cp Pfad/Quellordner Pfad/Zielordner
mv "Move" - verschiebt eine Datei Beispiel: mv Pfad/Quellordner Pfad/Zielordner
rm "Remove" - löscht Dateien Datei: rm Pfad/Datei; Ordner: rmdir -R ~/Documents/Löschen
mkdir "Make Directory" - erzeugt einen neuen Ordner Beispiel: mkdir "Ordnerbezeichnung"
touch Erzeugt eine leere Datei Beispiel: touch Dateiname(.Dateitypendung)
